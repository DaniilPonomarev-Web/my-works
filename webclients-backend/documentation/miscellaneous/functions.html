PaginatedResponseType
PaginatedResponseType(classRef: ClassType)

PaginatedResponseType<T> создает абстрактный класс, предназначенный для представления пагинированных данных в GraphQL. classRef - тип класса, который является частью пагинированного ответа (например, ClientDataDTO). Абстрактный класс PaginatedResponseTypeClass содержит массив объектов типа T (данные) и целое число (total), указывающее общее количество записей. Это полезно для отображения больших данных с поддержкой пагинации.

Parameters :
Name Type Optional
classRef ClassType No
